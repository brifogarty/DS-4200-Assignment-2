Firefox

About:

Mozilla Firefox, or simply Firefox, is a free and open-source web browser developed by the Mozilla Foundation and its subsidiary, Mozilla Corporation (via Wikipedia).

  • Website: Firefox Download
  • Developers: Mozilla Corporation, Mozilla Foundation
  • Initial release date: 2002
  • Stable release: 48.0, 51.0.1, 62.0
  • Programming languages: JavaScript, Cascading Style Sheets, C, C++, XBL, XUL
  • Engines: SpiderMonkey, Gecko
